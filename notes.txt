# ffwizard survey

- luci-mod-falter/luasrc/view/freifunk/index.htm
  - if installed and !runbefore, redirect to admin/freifunk/assistent
- assistant.lua click flow
  - prepare() via /assistant route
  - startWizard.lua
    - do available upgrades, then changePassword
  - changePassword.lua
    - changes root password immediately, then generalInfo
  - generalInfo.lua
    - changes and commits system and freifunk uci sections, then decide
  - decide.htm
    - pick optionalConfig or sharedInternet
  - shareInternet.lua
    - takes input for sharenet and bandwidth
    - changes ffwizard uci section without commit
    - then optionalConfigs
  - optionalConfigs.lua
    - sets ffwizard.settings.enableStats without commit, then wireless
  - wireless.lua
    - olsr.prepareOLSR, firewall.prepareFirewall
    - takes input for wifi mode, mesh ips, ssid name, dhcp ips
    - change firewall, olsrd, system, wireless, network, luci_statistics uci sections without commit
    - change dhcp, olsrd, network, ffwizard uci sections without commit
    - has cleanup code that runs for each wifi-device beforehand
    - then applyChanges
  - commit() via /assistant/applyChanges route
  - reboot.htm
  - reset() via /assistant/cancel route
